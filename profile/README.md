# TukeWIN

TukeWIN is a software development company focused on building reliable, production-grade systems for B2B clients. We specialize in full-stack applications and websites, covering the entire lifecycle from architecture and implementation to deployment and long-term operation.

## What We Do
- Full-stack web applications and custom websites
- Backend systems, APIs, payment gateways and data-driven platforms
- Frontend applications with modern frameworks
- Long-term system maintenance and production support
- Security-aware and performance-oriented engineering

## Technical Stack
- **Languages:** C, C++, PHP, JavaScript, TypeScript, Python  
- **Frontend:** React, React Native, Svelte, HTML5, CSS, Tailwind  
- **Backend:** Symfony, Laravel, Node.js  
- **Databases:** MySQL, PostgreSQL  
- **Other areas:** Systems programming, reverse engineering, security-focused development
